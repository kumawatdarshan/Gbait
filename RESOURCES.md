# GBA Emulator Resoucres

## Reference Manuals

- [Pan Docs](https://gbdev.io/pandocs/) GB/GBC bible
- [GBATEK](https://problemkaputt.de/gbatek.htm) GBA/NDS bible.
- [GBATEK: Rust markdown fork](https://rust-console.github.io/gbatek-gbaonly/)
- [ARM7TDMI TRM](https://documentation-service.arm.com/static/5e8e1323fd977155116a3129) pipeline and instruction timing
- [Copetti's GBA Architecture](https://www.copetti.org/writings/consoles/game-boy-advance/) systems-architecture essay, read before GBATEK
- [TONC](https://www.coranac.com/tonc/text/toc.htm) canonical GBA programming tutorial
- [The Audio Advance](http://belogic.com/gba/) GBA sound hardware with demos

## Community

- [emudev.org/system_resources](https://emudev.org/system_resources) curated links per-system, test ROMs, cycle-counting notes
- [gbadev.net](https://gbadev.net) Discord, resource list, getting-started guide
- [awesome-gbadev](https://github.com/gbadev-org/awesome-gbadev) curated resource list, community libraries
