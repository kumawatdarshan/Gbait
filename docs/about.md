# About This Project

**Gbait** is a GBA emulator built in Rust — a learning project, one session at a time.

## How It Works

- **Checkpoints** (daily) — one thing per session, 3 focus blocks, commit at the end
- **Milestones** (weekly) — defined each Monday or Friday, 5 checkpoints per week
- **Releases** (versioned) — ad-hoc, when enough milestones cohere into something shippable

## Weekly Rhythm

| Day | Focus |
|---|---|
| Mon–Thu | Feature implementation |
| Fri | Review, test, document |
| Sat | Architecture, refactoring |
| Sun | Testing, documentation, journal |
