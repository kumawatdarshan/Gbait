# Milestones

Milestones are defined **weekly** at Monday start or Friday review.
See [`PROGRESS.md`](./PROGRESS.md) for the live milestone map.

There are no pre-defined milestones. Each week's goal emerges from
what was learned the previous week. Releases (v1.0, v1.1, ...) are
declared ad-hoc when a collection of weekly milestones coheres into
a shippable whole.
